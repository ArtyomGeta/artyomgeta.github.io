Логическая функция F задаётся выражением (x → y) ∧ (y → z). На рисунке приведён фрагмент таблицы истинности функции F. Определите, какому столбцу таблицы истинности функции F соответствует каждая из переменных x, y, z.
Источник: ссылка

Программируемая часть задания довольно простая. Мы перебираем все возможные варианты с переменными и получаем результат, который позже нужно сопоставить с данной таблицей. (Вот это уже сложно :D).

НЕОБХОДИМО взять всё условие в скобки!
Запись if (): равна по значению if () == True: (аналогично if not(): равна if () == False:)

print('x', 'y', 'z')
for x in range(2):
    for y in range(2):
        for z in range(2):
            if ((x <= y) and (y <= z)):
                print(x, y, z)
